## Commands

| Command | Purpose |
|---------|---------|
| `npm run dev` | Local development |
| `npm run deploy` | Deploy to Cloudflare |
| `npm run cf-typegen` | Generate TypeScript types |

Run `npm run cf-typegen` after changing bindings in wrangler.jsonc.

## Do Not

- Never manually edit `worker-configuration.d.ts` — it is auto-generated by `npm run cf-typegen`
- No test/lint scripts exist in this project — do not guess or run non-existent scripts

## Must

- Run `npm run cf-typegen` after changing bindings in wrangler.jsonc
- Use `wrangler secret put <KEY>` for secrets — never hardcode in source or config

## TypeScript Formatting

2 spaces, single quotes, no semicolons, trailing commas.
